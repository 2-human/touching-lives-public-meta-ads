## Verification checklist (post-composition)

Per `css-isolation.md` §"Verifying isolation at composition time":

1. Page loaded without `?review=1` → no `[data-review-mode="on"]` on `<html>`; only the `.review-toggle-btn` visible; no `review-mode.css` or `review-mode.js` fetched.
2. Page loaded with `?review=1` → `[data-review-mode="on"]` present on `<html>`; banner + sidebar + entry-button (in narrow viewports) visible.
3. Hovering an anchored element reveals exactly one `.review-pill`; moving cursor away hides it; nested anchored elements show only the deepest one's pill.
4. Modal buttons render as widget variants (not host-page site buttons).
5. Every has-comment element carries the 1px dashed outline at low-alpha `accent-warm`.

Sanity-check by loading `/feed-preview.html?review=1` after the bootstrap is wired in.
