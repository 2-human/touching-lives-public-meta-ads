## Wiring

Every generated HTML page (`feed-preview.html` at hub, `details/{slug}.html` for each ad) loads two scripts in `<head>`:

```html
<script src="{rel_prefix}review.config.js" defer></script>
<script src="{rel_prefix}review-bootstrap.js" defer></script>
```

Emitted by `work-setup/build-feed-preview.py :: review_widget_scripts(rel_prefix)`. `rel_prefix` is empty for the hub and `../` for detail pages.
